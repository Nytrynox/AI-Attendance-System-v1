# AI Attendance System v1.0

Automated attendance management system using facial recognition with anti-spoofing detection and real-time reporting.

## Features

- Face detection and recognition with DeepFace
- Liveness detection to prevent photo-based spoofing
- Automated attendance logging with timestamps
- CSV and Excel report generation
- Web-based dashboard for attendance analytics

## Tech Stack

Python, DeepFace, Flask, OpenCV, SQLite

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Attendance-System-v1.git
cd AI-Attendance-System-v1
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```
